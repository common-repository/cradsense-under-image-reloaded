== Description ==

Places specified Adsense code under image(s) in a post.
If there are no images no Adsense code will be insterted. Also the ads will only appear in single post pages.
This refreshed plugin have more features than previous one.
In this one, you can:
1. Configure to have the image shown only on one image of multiple images on particular post.
2. Configure whether it's shown on first image, last image, or random.
3. If you choose to show on multiple images, you can determine how much images you want to have ads in it.


== Installation ==

1. Upload `cr-adsense-under-image-reloaded.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to options, select 'Adsense Under Image,' and change configs.
